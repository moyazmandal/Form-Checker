Form Checker

A simple JavaScript project that checks user selections from a form and displays the results on the webpage.

How It Works
Select the Subscribe checkbox if you want to subscribe.
Choose a payment method:
Visa
Master Card
PayPal
Click the Submit button.
The page will display your subscription status and selected payment method.
